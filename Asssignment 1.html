What is HTML & CSS?

HTML means Hyper Text Markup Language. It is used for creating webpages and it is created by using tags. Each tag has a specific purpose.

CSS means Cascading Style Sheets. It is the for describing the presentation of the web page. It involves styling web page through colouring, layout and even changing the fonts making it very good and presentable

HTML and CSS are two of the core technologies for building web pages.
